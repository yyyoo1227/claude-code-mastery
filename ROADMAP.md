# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요
HTML, CSS, JavaScript, TailwindCSS를 활용한 반응형 개발자 웹 이력서 개발

## 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3**: 스타일링
- **JavaScript**: 인터랙션
- **TailwindCSS**: 유틸리티 기반 CSS 프레임워크

---

## 개발 단계

### 1단계: 프로젝트 초기 설정
- [ ] 프로젝트 폴더 구조 생성
- [ ] TailwindCSS 설정 (CDN 또는 npm)
- [ ] 기본 HTML 템플릿 작성

### 2단계: 레이아웃 구성
- [ ] 헤더/네비게이션 섹션
- [ ] 히어로(소개) 섹션
- [ ] 기술 스택 섹션
- [ ] 경력 섹션
- [ ] 프로젝트 섹션
- [ ] 연락처 섹션
- [ ] 푸터

### 3단계: 스타일링
- [ ] TailwindCSS 클래스 적용
- [ ] 반응형 디자인 (모바일/태블릿/데스크톱)
- [ ] 다크모드 지원 (선택사항)

### 4단계: 인터랙션
- [ ] 스크롤 애니메이션
- [ ] 네비게이션 스무스 스크롤
- [ ] 모바일 메뉴 토글

### 5단계: 마무리
- [ ] 크로스 브라우저 테스트
- [ ] 성능 최적화
- [ ] 배포 (GitHub Pages)

## 이력서 콘텐츠 구조

### 프로필
```
이름: 홍길동
직함: 프론트엔드 개발자
소개: 사용자 경험을 중시하는 웹 개발자입니다.
```

### 기술 스택
```
- Frontend: HTML, CSS, JavaScript, React
- Styling: TailwindCSS, SCSS
- Tools: Git, VS Code, Figma
```

### 경력
```
ABC 테크 | 프론트엔드 개발자 | 2022.03 - 현재
- 웹 애플리케이션 개발 및 유지보수
- UI/UX 개선 작업

XYZ 스타트업 | 주니어 개발자 | 2020.06 - 2022.02
- 반응형 웹사이트 구축
- 협업 도구 활용한 프로젝트 관리
```

### 프로젝트
```
프로젝트 1: 쇼핑몰 웹사이트
- 기술: React, TailwindCSS
- 설명: 반응형 이커머스 사이트 개발

프로젝트 2: 포트폴리오 사이트
- 기술: HTML, CSS, JavaScript
- 설명: 개인 포트폴리오 웹사이트
```

### 학력
```
OO대학교 | 컴퓨터공학과 | 2016 - 2020
```

### 연락처
```
이메일: developer@email.com
GitHub: github.com/username
LinkedIn: linkedin.com/in/username
```
